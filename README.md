Features Included:
Dashboard UI: Clean, sidebar navigation, dark/light nuances.

Habit Tracker: Daily tick boxes.

Auto-Archiving: On page load, it checks if the date has changed. If a new day is detected, it saves yesterday's progress to history and resets the ticks.

Sticky Notes: A dedicated section for notes.

Visual Analytics: Uses Chart.js to visualize your habit consistency.

Data Export: A button to download your database as a .json file (Notepad compatible) for local backup.
